# Thich2Hand

[thich2hand.com](https://thich2hand.com)

Place to buy, exchange, and sell second hand.

DEV
============================

Build dev:
```shell
docker-compose build
```
Run dev:
```shell
docker-compose run
```

DEPLOY
============================

Build deploy:
```shell
docker-compose -f docker-compose-deploy.yml build
```
Run deploy:
```shell
docker-compose -f docker-compose-deploy.yml up
```


# Pre-commit
```
pre-commit install
pre-commit run
```
